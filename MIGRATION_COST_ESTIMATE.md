## 💰 토큰 비용 추정 (AI 에이전트 사용 시)

### 추정 가정
- 평균 토큰 비용: 입력 1K tokens ≈ $0.01, 출력 1K tokens ≈ $0.03
- 코드 작성: 파일당 평균 200-500줄
- 리뷰 및 수정: 각 파일당 2-3회 반복

### Phase별 토큰 비용

#### Phase 1: 환경 설정 (약 50K-80K tokens)
- **입력**: 프로젝트 구조 분석, MUI 문서 참조 (20K tokens)
- **출력**: 테마 설정 파일, 색상 토큰 변환 코드 (30K-60K tokens)
- **예상 비용**: $0.50 - $1.00

#### Phase 2: 공통 컴포넌트 (약 200K-300K tokens)
- **입력**: 
  - 각 wrapper 컴포넌트 코드 분석 (11개 × 5K = 55K tokens)
  - MUI 컴포넌트 문서 참조 (30K tokens)
- **출력**:
  - 컴포넌트 마이그레이션 코드 (11개 × 10K-20K = 110K-220K tokens)
  - 테스트 및 수정 반복 (50K tokens)
- **예상 비용**: $4.00 - $7.00

#### Phase 3: 페이지별 마이그레이션 (약 300K-450K tokens)
- **입력**:
  - 각 페이지 코드 분석 (9개 × 10K = 90K tokens)
  - MUI 컴포넌트 사용법 참조 (30K tokens)
- **출력**:
  - 페이지별 마이그레이션 코드 (9개 × 20K-30K = 180K-270K tokens)
  - CSS 파일 수정 (9개 × 5K = 45K tokens)
  - 테스트 및 수정 반복 (50K tokens)
- **예상 비용**: $7.00 - $12.00

#### Phase 4: 스타일 조정 (약 80K-120K tokens)
- **입력**: 전체 스타일 파일 분석 (30K tokens)
- **출력**: 스타일 수정 및 최적화 (50K-90K tokens)
- **예상 비용**: $1.00 - $2.00

#### Phase 5: 테스트 및 버그 수정 (약 150K-250K tokens)
- **입력**: 테스트 결과 및 버그 리포트 (50K tokens)
- **출력**: 버그 수정 코드 (100K-200K tokens)
- **예상 비용**: $3.00 - $6.00

#### Phase 6: 정리 및 문서화 (약 30K-50K tokens)
- **입력**: 프로젝트 전체 구조 (20K tokens)
- **출력**: 문서 업데이트, 코드 정리 (10K-30K tokens)
- **예상 비용**: $0.30 - $1.00
