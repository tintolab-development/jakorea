## 🎯 선별 기준

### 기준 1: 비즈니스 임팩트
- 🟢 높음: 핵심 사용자 플로우에 직접 영향
- 🟡 중간: 보조적 기능
- 🔴 낮음: 선택적 기능

### 기준 2: 사용자 경험 개선 효과
- 🟢 높음: 사용자 혼란을 크게 줄이는 화면
- 🟡 중간: 부분적 개선
- 🔴 낮음: 미미한 개선

### 기준 3: 구현 복잡도
- 🟢 낮음: 기존 컴포넌트 재사용 가능
- 🟡 중간: 새로운 로직 필요
- 🔴 높음: 복잡한 상태 관리 및 로직 필요

### 기준 4: 다른 화면에 대한 영향도
- 🟢 높음: 다른 화면의 기반이 되는 화면
- 🟡 중간: 독립적이거나 일부 의존성
- 🔴 낮음: 완전히 독립적
