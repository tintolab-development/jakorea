## 🔍 디자이너 검토 사항

### ⚠️ 발견된 문제점

#### 1. 색상 중복 및 유사성
- **스폰서** (`#ff7a45`)와 **정산** (`#fa541c`)이 모두 주황 계열로 유사함
- **일정** (`#fa8c16`)도 주황 계열로 3개 도메인이 비슷한 색상 사용
- 시각적 구분이 어려울 수 있음

#### 2. 색상 대비율 (접근성)
- WCAG 2.1 AA 기준: 텍스트와 배경 대비율 4.5:1 이상 필요
- 현재 Light 색상 배경에 Primary 색상 텍스트 조합 검토 필요
- Dark 색상 사용 시 가독성 확인 필요

#### 3. 브랜드 아이덴티티
- JA Korea 브랜드 컬러와의 일관성 확인 필요
- 교육 기관 특성에 맞는 색상 톤 조정 필요

#### 4. 색상 사용 가이드라인 부재
- 각 색상이 어떤 UI 컴포넌트에서 사용되는지 명확하지 않음
- 상태별 색상 (active, pending, completed 등) 매핑 불명확

## 💡 디자이너 제안사항

### 1. 색상 팔레트 재구성 제안

#### 옵션 A: 색상 계열 다양화
```
스폰서: 주황 계열 유지 (#ff7a45)
학교: 청록 계열 유지 (#13c2c2)
강사: 보라 계열 유지 (#722ed1)
프로그램: 파랑 계열 유지 (#2f54eb)
신청: 분홍 계열 유지 (#eb2f96)
일정: 노랑 계열로 변경 (#fadb14) - 주황과 구분
매칭: 녹색 계열 유지 (#52c41a)
정산: 빨강 계열로 변경 (#ff4d4f) - 주황과 구분
```

#### 옵션 B: 색상 그룹핑
```
기관 관련 (스폰서, 학교): 차분한 톤
  - 스폰서: 주황 계열 (#ff7a45)
  - 학교: 청록 계열 (#13c2c2)

인물 관련 (강사): 전문성 강조
  - 강사: 보라 계열 (#722ed1)

프로그램 관련 (프로그램, 신청, 일정): 활동성 강조
  - 프로그램: 파랑 계열 (#2f54eb)
  - 신청: 분홍 계열 (#eb2f96)
  - 일정: 주황 계열 (#fa8c16)

관리 관련 (매칭, 정산): 효율성 강조
  - 매칭: 녹색 계열 (#52c41a)
  - 정산: 빨강 계열 (#ff4d4f)
```

### 2. 접근성 개선 제안

#### 색상 대비율 검증
- 모든 Primary 색상과 Light 배경 조합의 대비율 측정
- WCAG 2.1 AA 기준 (4.5:1) 충족 여부 확인
- 미충족 시 색상 조정 또는 텍스트 색상 변경

#### 색상 의존도 감소
- 색상만으로 정보 전달하지 않기
- 아이콘, 텍스트 라벨과 함께 사용
- 상태 표시는 색상 + 텍스트 조합

### 3. 색상 사용 가이드라인 제안

#### UI 컴포넌트별 색상 적용 규칙

**Tag 컴포넌트**
- 도메인 식별: Primary 색상 사용
- 상태 표시: Ant Design 기본 색상 (success, warning, error 등)
- 예시: `<Tag color="blue">프로그램</Tag>` → `<Tag color={domainColors.program.primary}>프로그램</Tag>`

**Badge 컴포넌트**
- 상태 표시: Ant Design 기본 색상 사용
- 도메인 식별: Primary 색상 사용 (선택적)

**Card 컴포넌트**
- 배경: Light 색상 사용 (선택적)
- 테두리: Primary 색상 사용 (선택적)
- 예시: `<Card style={{ borderColor: domainColors.program.primary }}>`

**Button 컴포넌트**
- Primary 버튼: Ant Design 기본 primary 색상
- 도메인별 액션: 해당 도메인 Primary 색상 (선택적)

**Table 컴포넌트**
- 행 강조: Light 색상 배경 (호버 시)
- 중요 정보: Primary 색상 텍스트

### 4. 상태별 색상 매핑 제안

| 상태 | Ant Design 색상 | 사용 예시 |
|------|----------------|----------|
| 활성 (active) | `success` (녹색) | 활성 프로그램, 승인 완료 |
| 대기 (pending) | `processing` (파랑) | 검토 중, 대기 중 |
| 완료 (completed) | `default` (회색) | 완료된 항목 |
| 취소 (cancelled) | `error` (빨강) | 취소된 항목 |
| 비활성 (inactive) | `default` (회색) | 비활성 상태 |

**주의**: 도메인 색상과 상태 색상을 혼용하지 않기
- 도메인 식별: 도메인 Primary 색상
- 상태 표시: Ant Design 기본 색상

### 5. 추가 색상 Variation 제안
