**대상**: JAKorea CMS 대시보드  
**목적**: 자동화 시스템 관점에서 **정말 필요한 기능만** 추가

## 📊 현재 대시보드 분석

### 현재 구성
1. **기본 통계 카드 4개**
   - 교육 프로그램 수
   - 신청 수
   - 매칭된 프로그램 수
   - 등록된 강사 수

2. **최근 활동 목록 3개**
   - 최근 신청 (5개)
   - 최근 매칭 (5개)
   - 최근 일정 (5개)

### 문제점 (자동화 시스템 관점 - 핵심만)
- ❌ **액션 아이템 부재**: 담당자가 즉시 처리해야 할 작업이 명확하지 않음
- ❌ **월별 정산 현황 부재**: "월별 관리가 담당자 입장에서 깔끔" (인터뷰 핵심)
- ❌ **정산 승인 대기 표시 부재**: 승인 워크플로우가 중요한데 대기 건수 미표시

## 🎯 개선 방향 (핵심만 - 클라이언트 인터뷰 기반)

### 3. **최근 활동 통합** (현재 기능 개선)

## 🎨 제안: 대시보드 구조 개선안 (핵심만)

### **레이아웃 구성 (상단 → 하단) - 간소화**

```
┌─────────────────────────────────────────────────────────────┐
│ 1. 즉시 처리 필요 작업 (Alert Bar) - 최상단                 │
│    - 정산 승인 대기 N건 (클릭 시 정산 목록)                │
│    - 신청 검토 대기 N건 (클릭 시 신청 목록)                │
│    - 일정 중복 경고 N건 (클릭 시 일정 캘린더)               │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│ 2. 핵심 지표 (KPI Cards) - 1행 구성                         │
│    [월별 정산 현황] [이번 달 신청 수] [진행 중 프로그램]    │
│    (기존 4개 카드 중 일부 개선)                             │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│ 3. 최근 활동 통합 피드 (Activity Feed)                      │
│    - 최근 신청/매칭/일정/정산을 시간순으로 통합 표시        │
│    (현재 3개 카드를 1개 통합 피드로 개선)                  │
└─────────────────────────────────────────────────────────────┘
```

**제거한 항목 (과도한 기능):**
- ❌ 자동화 프로세스 상태 모니터링 (과도한 모니터링)
- ❌ 트렌드 차트 (6개월 추이 등 - 복잡함)
- ❌ 빠른 액션 버튼 (이미 사이드바에 있음)
- ❌ 예산 사용률 (예산 관리가 Phase 9인데 아직 구현 안됨)

## 📋 상세 기능 제안 (핵심만)

### **1. 즉시 처리 필요 작업 (Alert Bar)** ⭐ 최우선

**목적**: 담당자가 매일 아침에 열어서 "오늘 뭘 해야 하지?"를 바로 알 수 있어야 함

**표시 항목:**
- 🔴 **정산 승인 대기**: 승인 대기 중인 정산 건수 (클릭 시 정산 목록으로 이동)
- 🟡 **신청 검토 대기**: 검토 대기 중인 신청 건수 (클릭 시 신청 목록으로 이동)
- 🟠 **일정 중복 경고**: 중복된 일정 건수 (클릭 시 일정 캘린더로 이동)

**UI 구성:**
```tsx
<Alert
  type="warning"
  message="정산 승인 대기 3건"
  description="승인이 필요한 정산이 있습니다."
  action={<Button onClick={handleNavigate}>확인하기</Button>}
  showIcon
  closable
/>
```

**자동화 연동:**
- 정산 산출 완료 시 자동으로 알림 표시
- 신청 접수 시 검토 대기 알림 표시
- 일정 중복 감지 시 자동으로 경고 표시

---

### **2. 핵심 지표 카드 개선**

#### **2.1 월별 정산 현황 카드** ⭐ 인터뷰 핵심
```
┌─────────────────────────┐
│ 이번 달 정산 현황       │
│                         │
│ 총 정산액: 15,000,000원 │
│                         │
│ [대기: 3건] [승인: 5건] │
│ [지급완료: 2건]         │
└─────────────────────────┘
```

**표시 정보:**
- 이번 달 총 정산액
- 상태별 정산 건수 (대기/승인/지급완료)
- 클릭 시 월별 정산 관리 페이지로 이동

**인터뷰 근거:**
> "월별 관리가 담당자 입장에서 깔끔" (CLIENT_INTERVIEW_2024-12-19.md)

#### **2.2 이번 달 신청 현황 카드** (기존 개선)
```
┌─────────────────────────┐
│ 이번 달 신청 현황       │
│                         │
│ 총 신청: 45건           │
│                         │
│ [접수: 12] [검토: 8]    │
│ [확정: 20] [거절: 5]    │
└─────────────────────────┘
```

**표시 정보:**
- 이번 달 총 신청 수
- 상태별 신청 수 (접수/검토/확정/거절)
- 클릭 시 신청 목록 페이지로 이동

#### **2.3 진행 중 프로그램 카드** (기존 개선)
```
┌─────────────────────────┐
│ 진행 중 프로그램        │
│                         │
│ 활성 프로그램: 12개     │
│                         │
│ [이번 주 일정: 5개]     │
│ [다음 주 일정: 8개]     │
└─────────────────────────┘
```

**표시 정보:**
- 현재 활성 상태인 프로그램 수
- 이번 주 예정된 일정 수
- 다음 주 예정된 일정 수
- 클릭 시 프로그램 목록으로 이동

**기존 카드 유지:**
- 등록된 강사 수 (변경 없음)

---

### **3. 최근 활동 통합 피드** (현재 기능 개선)

**현재**: 최근 신청/매칭/일정을 별도 카드로 표시  
**개선**: 통합 타임라인 형태로 표시

```
┌─────────────────────────────────────┐
│ 최근 활동                           │
│                                     │
│ [2시간 전] 신청 접수                │
│   프로그램: 청소년 리더십 워크샵   │
│   주체: 서울고등학교                │
│   상태: 접수                        │
│                                     │
│ [3시간 전] 매칭 확정                │
│   프로그램: 온라인 과학 탐구 세미나 │
│   강사: 김강사                      │
│                                     │
│ [5시간 전] 일정 등록                │
│   프로그램: 창의적 글쓰기 강의     │
│   일시: 2024-12-20 14:00            │
│                                     │
│ [1일 전] 정산 승인                  │
│   프로그램: 디지털 리터러시 워크샵  │
│   금액: 1,200,000원                 │
└─────────────────────────────────────┘
```

**표시 정보:**
- 최근 24시간 내 활동 (신청, 매칭, 일정, 정산 통합)
- 시간순 정렬 (최신순)
- 각 활동의 핵심 정보 표시
- 클릭 시 해당 상세 페이지로 이동

**장점:**
- 현재 3개 카드가 차지하는 공간을 1개로 통합
- 시간순으로 정렬되어 최신 활동을 쉽게 파악
- 다양한 활동 유형을 한눈에 확인 가능

## 📊 데이터 요구사항 (최소한)

### **필요한 API 엔드포인트**

1. **대시보드 통계 API**
   ```typescript
   GET /api/dashboard/stats
   Response: {
     // 즉시 처리 필요 작업
     pendingActions: {
       settlementApprovalPending: number
       applicationReviewPending: number
       scheduleConflictCount: number
     }
     // 월별 정산 현황
     monthlySettlement: {
       totalAmount: number
       pendingCount: number
       approvedCount: number
       paidCount: number
     }
     // 이번 달 신청 현황
     monthlyApplications: {
       total: number
       byStatus: Record<ApplicationStatus, number>
     }
     // 진행 중 프로그램
     activePrograms: {
       count: number
       thisWeekSchedules: number
       nextWeekSchedules: number
     }
   }
   ```

2. **활동 피드 API**
   ```typescript
   GET /api/dashboard/activity-feed?limit=20
   Response: {
     activities: Array<{
       id: string
       type: 'application' | 'matching' | 'schedule' | 'settlement'
       action: string
       programName: string
       details: Record<string, any>
       createdAt: string
     }>
   }
   ```
