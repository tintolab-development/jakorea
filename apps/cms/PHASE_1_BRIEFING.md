## 📝 Mock 데이터 수정 가이드

Mock 데이터는 상황별로 수정 가능합니다:

### 수정 가능한 파일
- `src/data/mock/sponsors.ts` - 스폰서 데이터
- `src/data/mock/schools.ts` - 학교 데이터
- `src/data/mock/instructors.ts` - 강사 데이터
- `src/data/mock/programs.ts` - 프로그램 데이터
- `src/data/mock/applications.ts` - 신청 데이터
- `src/data/mock/schedules.ts` - 일정 데이터
- `src/data/mock/matchings.ts` - 매칭 데이터
- `src/data/mock/settlements.ts` - 정산 데이터

### 수정 방법
1. 각 파일의 배열 데이터 직접 수정
2. 데이터 개수 조정 (Array.from의 length 값 변경)
3. 상태 분포 조정 (status 배열의 확률 조정)
4. 관계형 데이터 일관성 유지 (프로그램-신청-매칭-정산 연결)

### 주의사항
- 타입 정의는 `src/types/domain.ts`에서 관리
- 관계형 데이터 수정 시 연결된 데이터도 함께 수정 필요
- Map 기반 조회 구조 유지
