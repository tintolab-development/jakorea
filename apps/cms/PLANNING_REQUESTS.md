# 기획 요청사항

## 화면 구성 다양화 요청

### 현재 상태

현재 화면 구성이 단조로워 개선이 필요합니다.

### 기획자 요청사항

1. **다양한 UI 컴포넌트 활용**
   - **팝업 모달 (Modal)**: 
     - 상세 정보 확인
     - 삭제 확인 다이얼로그
     - 빠른 정보 수정
     - 알림/공지사항 표시
   
   - **사이드 패널 (Drawer)**: 
     - 상세 정보 표시
     - 필터 패널
     - 설정 패널
     - 빠른 작업 패널
   
   - **드롭다운 메뉴 (Dropdown)**: 
     - 액션 메뉴 (수정/삭제/복사 등)
     - 상태 변경 메뉴
     - 내보내기/가져오기 메뉴
   
   - **토스트 알림 (Notification/Message)**: 
     - 작업 성공/실패 알림
     - 진행 상태 알림
   
   - **탭 (Tabs)**: 
     - 상세 페이지 내 정보 분류
     - 여러 뷰 전환
   
   - **아코디언 (Collapse)**: 
     - 정보 그룹화
     - 필터 옵션 접기/펼치기

2. **사용자 경험 개선**
   - **빠른 작업 (Quick Actions)**: 
     - 테이블 행에서 바로 수정/삭제
     - 일괄 선택 및 작업
     - 드래그 앤 드롭 (필요시)
   
   - **상태 표시**: 
     - Badge로 상태 표시
     - 진행률 표시 (Progress)
     - 아이콘으로 시각적 구분
   
   - **검색 및 필터 강화**: 
     - 고급 필터 패널
     - 저장된 필터
     - 검색 히스토리

3. **화면 구성 제안**
   - 목록 페이지: 테이블 + 사이드 패널 (필터/상세)
   - 상세 페이지: 탭으로 정보 분류 + 모달 (빠른 수정)
   - 폼 페이지: 단계별 진행 (Steps) 또는 탭으로 분리
   - 대시보드: 카드 그리드 + 차트 + 최근 활동

4. **인터랙션 개선**
   - 호버 시 추가 정보 표시 (Tooltip)
   - 클릭 가능한 요소 명확히 표시
   - 로딩 상태 시각화 (Skeleton)
   - 빈 상태 (Empty) 디자인

### 적용 예시

#### 목록 페이지
- 테이블 행 클릭 → 사이드 패널로 상세 정보 표시
- 액션 버튼 → 드롭다운 메뉴로 여러 작업 제공
- 필터 버튼 → 사이드 패널로 고급 필터 표시

#### 상세 페이지
- 탭으로 기본 정보/이력/관련 데이터 분리
- 빠른 수정 버튼 → 모달로 간단한 정보 수정
- 삭제 버튼 → 확인 모달

#### 폼 페이지
- 단계별 진행 (Steps)으로 긴 폼 분리
- 또는 탭으로 섹션 분리
- 저장 전 확인 모달

### 구현 우선순위

1. **Phase 2 이전**: 기본 모달, 사이드 패널, 드롭다운 메뉴
2. **Phase 2**: 탭, 아코디언, 토스트 알림
3. **Phase 3 이후**: 고급 필터, 단계별 폼, 드래그 앤 드롭

### Ant Design 컴포넌트 다양하게 활용 요청

**기획자 요청사항**:
- Ant Design에서 제공하는 모든 컴포넌트를 적극 활용하여 화면을 다양하고 풍부하게 구성
- 각 기능에 맞는 최적의 컴포넌트 선택
- 컴포넌트 조합을 통한 사용자 경험 개선

**활용 예시**:
- **Breadcrumb**: 페이지 경로 표시
- **Timeline**: 이력/히스토리 표시
- **Tree**: 계층 구조 데이터 표시
- **List**: 카드형 목록 표시
- **Avatar**: 사용자/엔티티 이미지
- **Tooltip**: 추가 정보 표시
- **Popover**: 상세 정보 팝오버
- **Popconfirm**: 확인이 필요한 액션
- **Alert**: 중요 알림 표시
- **Progress**: 진행률 표시
- **Spin**: 로딩 표시
- **Skeleton**: 로딩 중 스켈레톤 UI
- **Typography**: 텍스트 스타일링
- **Divider**: 섹션 구분
- **Space**: 요소 간격 조정

### 참고

- 1920x1080 고정 레이아웃에 최적화
- Ant Design 공식 문서: https://ant.design/components/overview
